Aspen (α)

Aspen is an online KonohaScript editor. You can try KonohaScript on your Web browser.

Aspen has following features:

  • Access your script from anywhere. Your writing is saved to a server storage.
  • Instant development. You can test your KonohaScript code instantly without download and install KonohaScript.

Aspen uses twitter OAuth authentication. If you don't have a twitter account, please create it before you sign in.
